Java Interview Questions

  • Q: What is Java?
    A: Platform-independent, object-oriented language.
  • Q: What is JVM?
    A: Java Virtual Machine runs bytecode.
  • Q: OOP concepts?
    A: Encapsulation, inheritance, polymorphism, abstraction.
  • Q: == vs equals()?
    A: == compares memory, equals compares data.
  • Q: What is constructor?
    A: Initializes object.
